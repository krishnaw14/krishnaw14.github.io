Ubisoft Game Jam 2018

Ubisoft Game Jam 2018 in Pune was a 48 hour game development hackathon where each team developed a game using a game engine (Unity, Unreal engine, etc.). The theme of the hackathon was to develop a 3D puzzle based treasure hunt.

The story of the game was that two different species (each represented by an individual) was stuck in a deserted planet and had to collect the respective orbs to get out of the planet. So, the game was essentially a multiplayer game in which 2 players competed against each other in a maze to collect treasure, which was in the form of orbs. The first player to collect the respective orbs would be the winner. A different feature of the game was that each player can collect the orbs of another player and release it some place else in the maze. Our focus was maily towards the gameplay side of the things.

We did not win but overall, it was my first exoerience with a game engine and learned a lot about gameplay and planning a game with proper detailing, so this was a good experience.

The final code can be found in the github repository: GameJam2018.
