Capturing semantic structures in Neural Machine Translations

This project was done under Seasons of Code, which is an inititative launched by Web and Coding Club IIT Bombay, along the lines of GSoC.

The project is in the area of Deep learning and neuro-linguistic programming and handles Neural Machine Translation with an unusual approach. We use the standard encoder-decoder architecture but wiht a modification, instead of using a single decoder, we have implmented multiple decoders, each of which can capture a different semantic structure of the sequence.

Right now, we are in the testing phase of the project and the initial phase looks good. More information about the results and codebase will be updated soo.
