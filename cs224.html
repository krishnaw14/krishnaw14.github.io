Computer Networking

There were three major assignments in CS224 (Computer Networking). The first one develop a client Server model to store key and values with the server being able to handle multiple clients at the same time. This was done using socket programming with event driven I/O using epoll. The second assignment was analysis of TCP with ns3 network simulator. In this assignment, we changed certain characterestics such as throughput, bandwidth, queue length, etc. and did the analysis of cwnd, queing delay and queue size with time using the trace file that is generated by the network simulator. The third assignment was the implementation of a distance vector routing protocol called RIP, which was essentially multiple iterations of the Bellman Ford update algorithm at every node.

The Final reports and codes can be found in the github repository: CS224-Assignments.
