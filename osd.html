Open Source Development

I like the concept of open source and have been an open source developer since August 2017. Starting with Enthought's Mayavi. Mayavi is a 3D plotting and scientific data visualization library in python. I corrected some bugs in the documentation and adding installation methods from various other sources such as edm, conda and conda-forge. Mayavi's owner is college professor, I helped close many redundant issues with him. With the aim of enhancing mayavi's functionalites, I also worked on multiblock readers for openfoam dataset, using XML reader for structured as well as unstructured grids.

I have also contributed to the repository Visma, owned by AerospaceResearch. Visma is a visual math solving library which provides steb by step visually dynamic solution to mathematical equations. I added more functions in the integration python script and also developed a script for differentiation of algebraic and trigonometric functions.

AIMACode is an organisation that implements the concepts covered in the book Artificial Intelligence: Modern Approach. I corrected a mistakes in the neural networks notebook in aima-python and also added pseudocode for backpropagation with regularization in aima-pseudocode. Apart from this, I added test cases and function for cross-entropy loss as though it was not explicitly mentioned in the original book but as the book cover the topic of KL divergence, it made sense to have cross-entropy loss.
